# IT Troubleshooting Guides

A collection of simple, practical troubleshooting guides for common IT support issues.

These guides are written to reflect real help desk and desktop support workflows.

---

## Included guides
- PC will not boot
- No internet connection
- Slow or unresponsive computer
- Printer not working

Each guide follows a basic structure:
- What to check first
- Common causes
- Simple steps to diagnose the issue

---

## Intended use
These guides are meant to support:
- Entry-level IT support
- Help desk troubleshooting
- Desktop support workflows

They focus on clarity, order of operations, and repeatable steps.
