# Printer Not Working

## Initial checks
- Confirm the printer is powered on
- Check paper and ink or toner levels

## Common causes
- Printer offline
- Driver issues
- Network connectivity problems

## Troubleshooting steps
1. Restart the printer
2. Restart the computer
3. Check printer status in system settings
4. Verify correct printer is selected
5. Reinstall or update printer drivers if needed
