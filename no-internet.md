# No Internet Connection

## Initial checks
- Confirm Wi-Fi or Ethernet is enabled
- Check for network outage
- Verify airplane mode is off

## Common causes
- Router or modem issue
- Network adapter disabled
- Incorrect IP configuration

## Troubleshooting steps
1. Restart the computer
2. Restart the router or modem
3. Check network adapter status
4. Try connecting to a different network
5. If wired, try a different Ethernet cable
